### Kyabetsu大头菜模拟器v0.0.4

============================

- 本模拟器为最初始版本，使用了集合啦！动物森友会的大头菜价格模型
- 周日上午5点到12点可以买入大头菜
- 之后周一到周六商店营业期间8:00-22：00可以询问大头菜价格，上午下午价格不同，分界线为12：00
- ※本周六如果有剩余大头菜没卖出去那么周日会坏掉，此时只能以一元一组（一般是100棵）的价格卖出
- ※bug会有很多，出现问题可以来杯咖啡

============================

- 初始化mana
- 查看当前mana
- 查看大头菜价格
- 买入大头菜 <数量>
- 卖出大头菜 <数量>
- 买入全部
- 卖出全部
